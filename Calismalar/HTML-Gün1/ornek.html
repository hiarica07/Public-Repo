What is Full Stack?

bodFull Stack Web Developer

Full Stack Web Developer is a person who can develop both client and server software.


In addition to mastering HTML and CSS,he/she also know how to:


Program a browser (e.g. using JS, jQuery, Angular, or Vuel)

Program a server (e.g. using PHP,ASP,Python or Node)

Program a databse (e.g. using SQL, SQLite, or MongoDB)
